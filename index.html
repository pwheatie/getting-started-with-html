The basics of the Web and Html

World Wide Web

It is a collection of html documents. The links between pages make it web like. Major pieces include your computer, the internet, and servers.

Additional Concepts:

Request through the internet use a protocol called http & servers are computers designed for file storage.

Four things about HTML

Text content is what your see, markup is what it looks like, references link to other documents and link to other pages.

Additional Concepts

Html markup contains elements that contain the opening tags and closing tags. In between these tags are the content. These tags are the basis of the html markup language.

Html Attributes

html attributes allow you to assign properties to html elements. For instance: "a href" is where a is an anchor tag and href is the attribute. The popular image element uses the attribute src to link images in web pages.

White Space

White space is critical for a properly designed webpage. Since Html will line up one element after another, a proper web designer must use white space to break up elements so the page is readable and aesthetically pleasant. A great video on the importance of whitespace can be seen here: Whitespace

Additional Concepts

Both Span and Div are elements used to produce whitespace. Span is an inline element, while Div is a block element.

Basic Html Vocabulary

Tag- Html tag is always contained by brackets
Element- and Html element refers to an opening tag and closing tag with items like attributes and text between the two tags.
Attribute- is a property of an Html element. href is an example.
